Програма: книжкова полиця

Книжкова полиця може містити книги або журнали, користувач може виконувати з ними наступні дії:

///////////////////////
// Меню консолі
+1 - Додавання
+2 - Видалення
3 - Запозичення з полиці
4 - Повернення на полицю
5 - Виведення списку наявних книг з можливістю сортування за вказаним параметром
    +1 - Сортування по автору
    +2 - Сортування по назві
    3 - Сортування по даті видання
    +4 - Сортування по кількості сторінок
    0 - Повернутися до попередніх пунктів
6 - Виведення списку наявних журналів з можливістю сортування за вказаним параметром
    +1 - Сортування по назві
    +2 - Сортування по кількості сторінок
    0 - Повернутися до попередніх пунктів
7 - Зберегти інформацію
9 - Показати полицю
0 - Вихід

///////////////////////
// Class Book
Властивості книги:
Назва*
Автор
Дата видання
Кількість сторінок*
Помітка, чи книга позичена з полиці*

///////////////////////
// Class Magazine
Властивості журналу:
Назва*
Кількість сторінок*
Помітка, чи журнал позичений з полиці*

Взаємодія з користувачем має відбуватись через термінал, вибір пунктів меню може здійснюватись через номери пунктів меню.
Вихід із програми має відбуватись через окремий пункт меню.
Книги та журнали мають зберігатись у файл (кількість файлів та формат - на вибір)
